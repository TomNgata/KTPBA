Step 01

DESIGN PRINCIPLES

A competitive bowling tournament platform demands high-impact data readability, energetic brand expression, and world-class ranking hierarchy.

Visual Personality

Deep-dark base with controlled neon energy. Inspired by modern esports tournament platforms — KTPBA Green + Red + Pink as the brand triad. Kenyan national colors drive the primary palette. Think: premium athletics meets competitive gaming.

High Contrast Brand-Led Competitive Energy
UI Density

Medium-compact. Leaderboard rows are tight for scanability (44px row height). Cards breathe more generously. The data must win — scores dominate visually. No decorative clutter; every pixel earns its keep.

44px rows Scores first
Competitive Data Readability
  • ① Scores in Bebas Neue — largest type element, instantly readable
  • ② Rank before name — left column, color-coded gold/silver/bronze
  • ③ Delta indicators — color arrows show movement at a glance
  • ④ Progressive disclosure — summary row → detail drawer
  • ⑤ Winner highlighting — background gradient on medal rows
Card System
  • Base card — rgba(255,255,255,0.04) + 1px border
  • Elevated card — rgba + backdrop-filter blur
  • Podium card — full gradient with medal glow
  • Match card — live pulsing border accent
  • Player card — gradient with avatar prominence

Step 08

MOTION & MICROINTERACTIONS

Purposeful motion reinforces competitive energy. All animations respect reduced-motion preferences.

Animation Tokens
NameValueUse
--dur-fast150msHover, focus states
--dur-base250msPage transitions, fades
--dur-slow600msScore bar animations
--dur-spring800msRank change, enter
--ease-outcubic-bezier(0,0,0.2,1)Elements leaving
--ease-springcubic-bezier(0.34,1.56,0.64,1)Score updates, bars
--ease-in-outcubic-bezier(0.4,0,0.2,1)Tab switches
Interaction Guidelines
  • 🏆 Score update — number flip animation, 600ms spring
  • 📊 Bar charts — bars grow from 0 on mount, 600ms stagger
  • 🔄 Rank change — row slides with translateY, green/red flash
  • 🎯 Bracket winner — border glow pulse, 300ms
  • 🃏 Card hover — translateY(-2px) + border brighten, 150ms
  • 📑 Page switch — fadeIn + translateY(6px→0), 250ms
  • 🔴 Live dot — opacity pulse 1.2s infinite ease-in-out
